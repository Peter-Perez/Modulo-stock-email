Product Low Stock Notification

Product Low Stock Notification is usefull to give the user update of product's on hand quantity or forecast quantity via mail.User can get list of products with their current quantity and their required quantity.Easily define low stock level global for all products or individual product. On basis of type of mode it will generate email if any product low stock.

Key Features:

  • You can get update by email on low stock of product.
  • You can set limit of minimum quantity as global for all the products or individual for every product.
  • You can set on which type of quantity you want to send notification.
  • On hand quantity or Forecast.
  • You can set or create a user whom you want to send notification as email.
  • You set the how many times in a hour,day,week or for a month you want to send email notification.
  • Cron will send notification in defined time automatically.

How To Use


Once module is installed you can see Product Stock Notification in settings of inventory, now you can set the functionality.

How to set product stock notification

How to set minimum quantity as a global for all the products

You just have to select Global for all products in Minimum quantity based on and set the minimum quantity in Quantity limit.


How to set minimum quantity as individual for all the products

You just have to select Individual for all products in Minimum quantity based on.


You just have to open product variants and set minimum quantity in Minimum Quantity field.
